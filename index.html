What I Use
React/React Native

A JavaScript library for building user interfaces.
Developing both web pages and mobile
application, this language takes advantage of the
Virtual DOM for efficient rendering, thus leading to
a more precise and ergonomic applcation.

RESTApi/GraphQL

A software intermediary that allows two
applications to communicate. In every website, it is
crucial that the website is able to communicate
with its servers and database, thus creating a better
User Interface and User Experience.

C#/.NET

An object-oriented programming language for Web
Development which requires the .NET framework. It
is crucial in every application to be able to run any
service and database quickly and efficiently, thus
improving on the experience of the user.

Node.js

A cross-platform environment that executes
JavaScript outside of the browser. This allows
proper and efficient testing for client-side scripting.
Combining with Express, web-building
and API's integrate in a smoother transition.

MongoDB

A cross-platoform DOM program that maps data to
a JSON object. Storing this data as flexible data
allows the querying and indexing easier for the
development of the server and can be improved
over time.

SQL

A language that accesses and manipulates data.
Being able to simply manipulate data allows the
developer to efficiently delegate the code more
effectively. Combining with Postgres allows
proper management of data.
